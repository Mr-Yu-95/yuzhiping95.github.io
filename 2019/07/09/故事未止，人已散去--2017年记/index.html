故事未止，人已散去—-2017年记

​ 2017.2.4《路还很长，故事还在继续》记录了2016年的点滴，在开篇第一段这样写道“拖延症犯起来是真的要命,本来打算年前写完的总结,拖到了现在.感觉我要是再不写，我都要写2017年的总结。”可现在呢。

​ 2017真的好难、好苦、好累，但却又好笑、好美、好幸运。如果你问我对生活的理解，我会说“生下来，活下去！”曾经看过一句话，内容大概是；生日时，谁给你的礼物最好，我说是我妈，因为我妈送给我一条命，到现在我还用着，活着比什么都重要。

​ 2017坚持很重要，放弃某件事远比你坚持下来要难的多。努力了那么久，真的忍心放下你的梦想？喜欢了那么久，真的忍心放弃喜欢的人？坚持最起码还会给我们一个目标，放弃往往写着悲伤和无奈。想要放弃却又不知后来该如何选择，何不在坚持一会儿，真正值得的东西都不会那么轻易，所谓希望也许就是付出努力有可能比完全放弃强那么一点点。

​ 2017年1月，新年第一天，按照传统，在00：00截下了2016年同时刻的说说进行发表作为新年的开场白。1月5号，坐公交去火车站，写了一篇说说，内容的结尾大概是这样的“在此立下flag，重擂战鼓，向过去宣战”，我建议大家不要立flag，这一年，“懒惰，自负，浮躁”这些代表过去的小婊子没少上门找我，还一副大哥过来玩呀 的样子。

​ 这个段落要留给老大，其实这份感谢本应写在2016，可偏偏发生在2017，这也恰好成为了2017可以拿的出手的骄傲。19号-24号这是一个能够让我吹一年的经历，我被济南的小姐姐包养了5天，真的很感谢那段时间叔叔阿姨的热情招待，真的很感谢你们的信任，这简直是我的幸运，希望你们的善意能给你们带来好运。其实大家可能不知道，阿姨家米饭特别好吃，哈。还记得打美赛期间最经典的一句话便是“我好想有一个脑子”。全程偷着划水的感觉真刺激，还好两位小姐姐当时认真能干，在此给两位小姐姐打call，最后总算是生拉硬拽着我这条废鱼熬了个通宵翻译完了任务，我当然还是负责吐泡泡，小弟在这给老大递可乐，老大记得喝完把瓶子给我。其实还有个问题问老大，叔叔阿姨不在家，你晚上睡觉不锁房门，你都不害怕么。。。。。。。

​ 2017年2月，更多的是有关过年的那些事，穿新衣，放鞭炮。其实过年早已没有以往那份欢乐，之所以还那么被大家重视，也许是因为这个节日能陪亲人聊聊天，看看电视，包包水饺，时间真的很快，亲人的伤病更甚至离去，会让你突然觉得你在时间面前真的无能为力，没有什么比亲人都在身边更令我感到安全的了，哪怕他们什么都不会，哪怕他们一辈子连自己的小县城都没出过，可他毕竟还在这个世上陪着你，你我还有归途。如果把他们的生命归于我的时间表中，我的时间真的不多了，也许你我最大的人生目标就是让父母过得好。

​ 2017年3月，据说每刮一阵风，都会有大雪从四面八方纷飞而来。

​ 2017年4月，暴风雪接连不断，天气始终没有好转，最终大雪压断了桥，人离开了。用张爱玲写的几句话描述再恰当不过了，海上月是天上月，眼前人是心上人，向来心是看客心，奈何人是剧中人。

​ 2017年5月，因为要跑一些文件，有幸在这个忙碌的月份回家偷懒，可老话说的好，欠下的债总是要还滴，随后的十几天，感觉天天都在开会，写报告，做总结，填表。当时还假惺惺的对自己说：这些多耽误我复习的时间等等。。。可看看后半年复习的鬼样子，果然是自己想太多了。有一篇网络文章伴随了我整个下半年，文章的开头是：纽约时间比加州时间早三个小时，但加州时间并没有变慢……你没有领先，你没有落后，在你自己的时区里，一切安排都准时。它后来出现的月份是9月，11月和18年1月。

​ 2017年6月，浑浑噩噩的混到了期末复习周，波哥结课时说：我们从3月开始，18点40上课天都黑透到现在太阳还没落山，我们看完了张艺谋2008奥运开幕纪录片等等。如果你们觉得我想表达一下抒情那就大错特错了，我现在都忘不了39页A4纸5号字的IT项目背诵材料，整个复习周基本都是在背这个宝贝，真是谁用谁知道啊。

​ 2017年7、8月，飘过期末考当天，外面下着大雨，也不知道是不是把脑子淋坏了，可是话说回来了，鱼为啥怕水呢，也许是烤鱼的缘故吧，双脚随性踏入steam坑，而且还是走在饥荒的路上，饥荒了三四天，总算把创实那几个宝宝等来了，搬搬行李，安顿好，教训就是搬行李是不会产生啥故事的，除了能进女生宿舍楼，当晚随性坑了紫薇学长一把，毕竟人丑家穷混吃才是王道，雪姐总是招黑的。回家缓了七八天，正式踏入了考研（颓废）的道路，从此越陷越深。整个7，8月应该是考研期间最坚挺的两个月，早饭老干妈配燕麦片，燕麦片喝着感觉就像浆糊一般，就这玩意儿我喝了两个月，至今还有半袋，忘了以前从学校带回家什么吃的，扔给狗，狗都不吃。。。另外事实证明上一天数学真的恶心，白天上课，晚饭恶心，晚上上自习接着学数学，半夜热到睡不着，这重要的两个月给我的最大收获就是懂得闭嘴努力，在每一个值得珍惜的日子里，拼命去成为自己想成为的人。月末，和两位大佬去世贸佰烧吃了下自助，当然有紫薇学长，也算给这两个月收了个尾。

​ 2017年9月开初，打着一切都是为了学习得幌子，晚七点，在违背物业规定的情况下，偷偷溜进逸夫楼，跟大爷打着游击，走地下车库，望风，蛰伏待机，互相配合，最终被大爷活捉现行，还好大爷好说话，放了我们。而我们此时此刻进行的正是轰轰烈烈的考研占位风波，无奈之下，一行人在楼下商讨对策，遂决定和宿管大爷商量，明早提前开门，未曾想大爷恪尽职守，拒绝让步，无奈之下，同泽帅商量一番决定晚上睡在基建部，早起站等逸夫开门。

​ 下面进入现场直播，北京时间六点整，各路英雄豪杰早已摩拳擦掌准备抢占603&604高地，这是一次演习和实战的完美融合，是学生们集体智慧的结晶，好，让我们回归现场，只见预热不及两分钟，场地外的选手早已按捺不住激动的心情，大爷担心自己珍贵的玻璃门，遂决定比赛提前开始。

​ 好，比赛开始了，开始了。卧槽，卧槽，卧槽，比赛结束了。让我们用慢镜头回顾一下这难以想象的比赛，只见大爷把门打开的一刹那，选手们飞奔而入，大爷紧紧的抱住门，别怕，门，我在。选手们在四楼，五楼，六楼各自为营，让我们看一下右上角的时间，42秒，42秒，选手们在负重的情况下以极短的时间爬上6楼，大家看这里灵活矫健的胖子，对这是我们翔书记，还有这里，大步向前的杜班长，还有这，这腿短的货竟然背着10斤的书包进行比赛，让我们看一下这到底是哪位仙人，等等，这tm不是我么。

​ 让我们进行一下赛后跟踪采访，哟，不赖嘛，我旁边竟然来了一位小姐姐，据本台独家获悉，这位小姐姐后来送过面包，橘子，柚子，雨伞等等，高挑身材皮肤好，端庄举止气质佳。让我们继续跟随摄像机的镜头，选手们脸上露着开心和喜悦，嘻嘻哈哈声不绝于耳，本台采访到此也基本结束，让选手们好好享受这份来之不易的喜悦，期待他们梦想成真，这里是烤鱼传媒，期待下次以更优质的服务与大家见面。

​ 另外，九月发生过自考游击部队与我正规军关于主权问题的摩擦事件，此事件毋庸置疑，主权方面不容侵犯，犯主权者，必驱之。

​ 九月结束之际，那位我一直视为标杆的小姐姐成功保研到浙江大学，恭喜小姐姐，她真的属于那种不声不响但是什么都能做的很好的人，一直以来安静的优秀，沉默得像成熟的稻子，这或许正是优等生和我这样伪优生的本质区别，假的终究是假的。总之，希望她继续好，to make each day count。

​ 2017年10月，9月份我们见证了朱日和练兵场的盛大阅兵，正所谓强国先强军，军强方能定邦，1号国庆之际，我们通过了国歌法，18号召开中国共产党第19次全国代表大会，本次盛大会议对考研狗来说相当重要，考点都在习总书记的话里，事实证明，习总重点真的好使。建国至今，我们见证了中国的飞速发展，或许我们还有很多不足很多缺陷甚至弊端，但这却能让我们丢到浮躁丢掉锐气，不忘初心，砥砺前行。

​ 2017年11月，背政治、背政治、背政治，背专业课，背专业课，背专业课，一副对联形容一下，上联：万水千山总是情，少背一点行不行？下联：天若有情天亦老，背的太少考不好！横批：乖乖去背，其实复习后半程每天最开心的就是和翔书记去食堂吃饭，不是因为食堂多么好吃，而是那十几分钟可以不用学习,这或许就是我学不好的原因，本月更有趣的是，在11月11号这天，出门现场确认，满大街的情侣，手挽着手，冷冷的狗粮在脸上胡乱的拍。

​ 2017年12月，空气中的紧张气氛陡然提升，其实对于考研大军来说，日期在这群人眼里显得没那么重要，倒计时早已充当了本时间段的时钟，10号的公考，16号的四六级考，23号的研考，从11月早已看清本人的成绩会是如何的惨败，所以也缺少了几分紧张与不安，每天看的最多的也还是政治和专业课，其实拿到数学试卷，等到专业课和政治分数，才明白舍不得为难自己，生活就会为难你这句话，我总是习惯性选择安逸，过没有压力的生活，可很多事，当选择放过自己，机会也同样会放过我，偷过的懒，会成为打脸的巴掌，等到调剂时，也才清楚在开始的时候舍得花力气去填平坑洼，未来的道路才能平坦畅通。研考过程出现了一些不愉快，但最后也算平稳解决。生活处处有惊喜，考点竟是3月光临的省实验，考试头天头科，更意外的和青青同学左右位，隔道相望，何等的猿粪，呵呵。考这些科目唯独专业课，最后10几分钟剩下的也不会，看看济南大屌开心的笑了。12月的尾声，一天一顿馆子饭，也谢谢各位大佬接济穷苦的我。

​ 我2017的故事讲完了，该讲讲周围的故事了。

​ 2017年他们走了，2018年也不会再回来了。黄大年，李佩，周有光，任新民，屠善澄，刘宝琛，陈学俊，申泮文，张忠培，柯俊，朱英国，南仁东，朱显谟，吕必松，高伯龙，余光中，童志鹏等等，他们悄悄的离开，没有打扰公众半分，太多的人对他们一无所知，然而他们却是当之无愧的大国脊梁，也是最不该被遗忘的存在，想起一句话，记住我的好或记住我就好，愿英雄们一路走好。

​ 回望过去的一年，我们经历过很多不开心的事。看到送外卖因为超时在电梯里哭出来的小哥。看到在大雨里嚎啕大哭的快递员。看到地铁口十二点多还在摆摊的麻辣烫老板。看到清洁工阿姨舍不得吃十五块钱的盒饭。我们曾因某些事情的发生而义愤填膺，也曾为了一些正能量的伸张而努力贡献自己的力量。尽管生活中，我们总是过得普普通通，没那么好也没想象中的那么困难，每个不开心的时刻，跟朋友聊天吐槽也总能甩掉那些阴霾。

​ 2017不过是人生这条路上的一个驿站，或许2018会很忙更累，日子会有些难过，生活也总是困难。但那些美好就像我们曾经为之感动的温暖一样。它可能会迟到，但不会让你久等，未来远比我们的过去更美好。

​ 岁月其实是一场有去无回的旅行，好的坏的都是风景。
